Honors Core

The honors core consists of six introductory classes that Turing students take during their freshmen and sophomore years. Turing students then choose additional honors CS electives, as well as department-wide CS electives. 

You can find a typical four-year honors degree plan on the UTCS degree plans page.

C S 314H: Data Structures

Students are introduced to advanced data structures and trained in software engineering techniques such as test methodology and agile workflow. Students complete a set of diverse and challenging programming projects, such as building a basic search engine or building a basic genetic algorithm to play Tetris.

Topics include: Java, object oriented programming, programming techniques, debugging, algorithm analysis, stacks, queues, linked lists, recursion, sorting, trees, hash tables, balanced trees, priority queue, graphs

C S 311H: Discrete Mathematics for Computer Science

Students are taught how to think mathematically and write proofs. Students are exposed to the fundamentals of Computer Science theory, such as graph theory, combinatorics, and mathematical induction. They learn to express problems mathematically and formally reason about computer science phenomena. Students are also exposed to automated theorem provers, and modern tools programmers use to reason about their program’s correctness formally.

Topic Include: Proofs, Set Theory, Number Theory, Graph Theory, Combinatorics, Induction, Contradiction, Recursion, Automated Theorem Provers, Master Theorem

C S 429H: Computer Organization and Architecture

Students learn about how a computer works - from transistors and logic gates to the compiler level - and how hardware and software work together. This is an intensive project based class in which, from scratch, students write a pipelined architecture in Verilog (a popular Hardware Design Language). Students are also taught C in this class as only prior knowledge of Java is assumed. Additionally students are required to blackbox test all projects and are responsible for writing clear efficient code to this end. Papers corresponding to particular projects are read and discussed. The class includes a capstone small group project where students are expected to build on a class project in novel ways.

Topics Include: Transistors, Logic Gates, Logic Diagrams, Processor Diagrams, Compilation, Thread Contexts, ISA Construction, Pipelining, Super Scalar, Cache Coherency, Atomics

C S 439H: Principles of Computer Systems

Students learn the building blocks of modern operating systems. They build a monolithic kernel from scratch in individual projects. Additionally they read relevant systems papers to enhance their understanding of why certain systems were developed the way they were.

Students learn to black and white box test each component of an operating system. They are assessed on the quality of their code as well as the novelty of their approach.

The class includes a final capstone class project, where the entire class works to build a fully functional operating system with graphics, sound, terminal interfaces etc. Students define their own standards and practices and rely on the faculty only when they encounter significant difficulty with a particular operating system concept. Generally this class is taught in C++ with a bit of Rust sprinkled in.

Topics Include: Threads, Processes, Synchronization Primitives, File Systems, Virtual Memory, Page Caches, Heap, Syscalls, fork and exec, Elf-Loading

C S 331H: Algorithms and Complexity

Students learn fundamental algorithms and complexity theory. This course is heavily proof-based and focuses on theory. Topics covered include dynamic programming, graph theory, and complexity theory. This class includes both a mathematical discovery of algorithms and the application of such techniques to code.

Topics Include: Dynamic Programming, Interval Scheduling, The Knapsack Problems, Path Finding, Max Flow, Algorithm Analysis, Complexity Analysis, Probabilistic Algorithms

C S 178H: Introduction to CS Research

This course helps students make the transition from classroom learning to the type of faculty-advised independent learning that is necessary to complete an honors thesis. This class gives students concrete advice about how to start researching with professors at UT, and offers practice and hands-on experience with the mechanics of research at the undergraduate and graduate level.

Topics Include: The research Process, Academic Reading and Writing, Graduate School, graduate school applications, the trajectory of an academic career, the role of research in industry, road to the Turing thesis.
